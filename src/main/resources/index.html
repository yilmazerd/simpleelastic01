Instructions to use this app

You can simply call this URL with any of the HTTP request methods

You can customize the response of the app by changing headers in your request. Following headers are used to customize the response

# Header Note Example
1 responseCode Status Code of the response 200,202,500 etc
2 responseDelay Additional delay to the response in milliseconds. 3000, 20, 500 etc. Must be integer
3 urlHeader URL header of the source of the response. If the client does not provide this header then this website would return. https://tools.learningcontainer.com/sample-json.json
4 contentType Content type of the response text/html; charset=ISO-8859-1 OR application/json

curl --location --request GET 'http://headermockserver-env.eba-enq6b9ib.us-east-1.elasticbeanstalk.com' \

--header 'responseCode: 504' \

--header 'responseDelay: 10' \

--header 'urlHeader: https://tools.learningcontainer.com/sample-json.json' \

--header 'contentType: application/json'

Or use a query

curl --location --request GET 'http://headermockserver-env.eba-enq6b9ib.us-east-1.elasticbeanstalk.com/?responseDelay=1000&responseCode=504&contentType=application/json&urlHeader=https://tools.learningcontainer.com/sample-json.json'

If the client has a variable in both header and in URL param, app will use the value in header and ignore the query parameter.
