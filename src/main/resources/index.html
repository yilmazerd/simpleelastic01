Instructions to use this app

You can simply call this URL with any of the HTTP request methods, GET, POST, PATCH

You can customize the response of the app by changing headers (or query parameters) in your request. Following variables are used to customize the response

# Header/Query parameter Note Example
1 responseCode Status Code of the response 200,202,500 etc
2 responseDelay Additional delay to the response in milliseconds. 3000, 20, 500 etc. Must be integer
3 urlHeader URL header of the source of the response. If the client does not provide this header then this website would return. https://tools.learningcontainer.com/sample-json.json
4 contentType Content type of the response text/html; charset=ISO-8859-1 OR application/json
5 responseContent Response. For Json, remove line breaks {"name":"john","age":22,"class":"mca"}

Pass the response data in header or in query parameters

curl --location --request GET 'http://mock.instantfunction.com/?responseDelay=30&responseCode=200&contentType=application/json&responseContent={"name":"john","age":22,"class":"mca"}'

use this app to remove line breaks

https://www.textfixer.com/tools/remove-line-breaks.php
